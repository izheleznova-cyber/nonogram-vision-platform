# JavaScript Format

## Переменная d

Большие кроссворды содержат встроенный JavaScript

var d=[...]

### Структура

d = [
    [a,b,c,d],
    ...
]

Каждый элемент состоит из четырех целых чисел.

Расшифровка каждого блока d[x]:
Поле
Формула
Значение
d[x][0] - Ia[0] - 1  начальная колонка           col_start
d[x][1] - Ia[1]      ширина блока                width
d[x][3] - Ia[3] - 1  строка                      row
d[x][2] - Ia[2]      цвет (0=пусто, >0=цвет)     color


Пример: Если блок имеет параметры [10, 5, 2, 3] и Ia = [1, 1, 1, 1], то:
Колонка: 10 - 1 - 1 = 8
Ширина: 5 - 1 = 4 (колонки 8, 9, 10, 11)
Строка: 3 - 1 - 1 = 1
Цвет: 2 - 1 = 1
Значит, в строке 1, колонках 8-11 будет цвет 1.
